# 🎓 Travaux Pratiques - Groupe 7 (2025-2026)

Bienvenue sur le dépôt officiel du **Groupe 7**. Cet espace centralise l'ensemble de nos projets et travaux pratiques d'ingénierie et de programmation structurée sous forme d'arbre.

---

## 📂 Structure du Dépôt (Arbre des Projets)

Chaque sous-dossier de la branche principale représente un projet ou un TP indépendant et complet :

* **`Gestion_Etudiants/`** : Application en langage C dédiée à la gestion dynamique et à l'organisation de données étudiantes (structures, pointeurs et allocations).
* **`Produit_Scalaire/`** : Programme d'implémentation de structures de données pour le calcul vectoriel et l'analyse de produits scalaires.

---

## 🛠️ Technologies Utilisées

* **Langage :** C (Norme C11 / C99)
* **Gestionnaire de version :** Git & GitHub
* **Environnement recommandé :** GCC / MinGW ou tout IDE compatible (VS Code, Code::Blocks)
